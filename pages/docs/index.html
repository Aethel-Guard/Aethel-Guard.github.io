P1-P7 Redlines

Hardcoded security constraints enforced on every request and response — cannot be bypassed.

  • P1: No PII exfiltration
  • P2: No jailbreak bypass
  • P3: No model substitution
  • P4: No silent failover
  • P5: No unapproved external calls
  • P6: No data retention beyond session
  • P7: No reduced-thinking execution
